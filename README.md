# 필리핀 부동산 렌탈 웹사이트 프로젝트

## 프로젝트 개요
필리핀에 거주하는 외국인(한국, 중국, 일본, 영어권)을 대상으로 하는 부동산 렌탈 검색 웹사이트

## 주요 기능
- 로그인 불필요한 공개 웹사이트
- 지역별, 유형별, 가격대별 매물 검색
- 다국어 지원 (한국어, 중국어, 일본어, 영어)
- 관리자 모드를 통한 매물 관리

## 대상 사용자
- **일반 사용자**: 필리핀 거주 외국인 (임차인)
- **관리자**: 매물 등록/관리 담당자

## 매물 유형
- House (단독주택)
- Condo (콘도미니엄)
- Village (하우스 빌리지)

## 주요 지역
- Manila (마닐라)
- Cebu (세부)
- Davao (다바오)
- Boracay (보라카이)
- Baguio (바기오)

## 기술 스택 (예정)
- **지도 서비스**: Google Maps API (무료 할당량 활용)
- **데이터베이스**: 미정
- **프론트엔드**: 미정
- **백엔드**: 미정

## 개발 단계
1. ✅ 요구사항 분석 및 기능 정의

## 참고사항
- Google Maps API 무료 할당량: 28,500회/월
- 소규모 부동산 사이트에 충분한 용량
- 추후 사용량 증가 시 다른 옵션 검토 예정
